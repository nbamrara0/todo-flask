# Flask Todo App

A simple Todo web application built using Flask and SQLite.

## Features

- Add tasks
- Update tasks
- Delete tasks
- Store tasks using SQLite database
- Simple Bootstrap UI

## Technologies Used

- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML
- Bootstrap
